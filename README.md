# 프로젝트 목적

- PPT 강사로도 출강하는 경우가 있는데, 이 때마다 지금까지 제작한 PPT의 예시를 보여주기 위해 각각의 파일 열어서 보여주는 것에 불편함을 느낌.
- 따라서, PDF 형태의 파일을 나열하여 보여줌으로써 조금 더 짧은시간에 많은 내용을 전달하고자 함.

# 프로젝트 기능

1. PDF 형태의 PPT 보여주기
2. PDF 상세보기(모달)
   - 주제
   - 제목
   - 디자인 컨셉
   - 만든 날짜
   - 발표 일자
   - 관련 과목
3. 배경 GIF
   - 너무 번잡하여 Simple한 이미지로 변경
